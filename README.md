## Sobre o Projeto

O Snitap é uma landing page de alta performance que explora o estado da arte do CSS moderno. O foco principal foi a criação de uma experiência imersiva através de **Motion Design**, utilizando técnicas como:

- **Scroll-Linked Animations**: Animações disparadas pelo scroll do usuário utilizando `animation-timeline: view()`.
- **Mosaico com CSS Grid**: Layout da galeria estruturado com `grid-template-areas` para um design assimétrico e responsivo.
- **Micro-interações**: Efeitos de hover refinados, animações de "bounce" calculadas manualmente e transições de gradientes animados (`background-size: 400%`).
- **Arquitetura Modular**: CSS organizado em módulos (BEM/SMACSS-like) para facilitar a manutenção e escalabilidade.

## Como rodar o projeto localmente

Para clonar e executar este projeto em seu computador, você precisará do Git instalado.

Passo a passo:


# Acessar a pasta do projeto

```
cd snitap-motion-landing-page
```

Após isso, basta abrir o arquivo `index.html` no seu navegador de preferência.

# Snitap — Skates with Style (EN)

## About the Project

Snitap is a high-performance landing page that explores the state of the art in modern CSS. The main focus was creating an immersive experience through **Motion Design**, using techniques such as:

- **Scroll-Linked Animations**: Animations triggered by user scroll using `animation-timeline: view()`.
- **CSS Grid Mosaic**: Gallery layout structured with `grid-template-areas` for an asymmetrical and responsive design.
- **Micro-interactions**: Refined hover effects, manually calculated "bounce" animations, and animated gradient transitions (`background-size: 400%`).
- **Modular Architecture**: CSS organized into modules for easier maintenance and scalability.
